# Potok Player

Potok Player is a React Native application that demonstrates a custom native integration of [libtorrent](https://libtorrent.org/) using Expo Modules. It allows you to download torrents directly within the mobile application, tracking progress and status natively on iOS and Android.

## Features

- **React Native & Expo**: Built with modern Expo tooling and React Native framework.
- **Native Libtorrent Wrapper**: Includes a custom Expo module (`modules/libtorrent`) that bridges C++ `libtorrent` functionality to JavaScript.
- **Magnet Link Downloading**: Start torrent downloads by passing a magnet link from JavaScript.
- **Progress Tracking**: Real-time event emission for torrent download state and progress to React Native UI.

## Project Structure

- `App.js`: Main entry point demonstrating the download functionality, event listeners, and basic UI layout.
- `modules/libtorrent`: The custom Expo module wrapping the native libtorrent implementation.
  - `ios/`: Contains the iOS Swift bridged implementation, Objective-C++ wrappers, and `.xcframework` dependencies.
  - `android/`: Contains the Android Gradle setup and JNI/Java integrations for libtorrent.
- `app.json` / `package.json`: Expo and npm configuration.

## Getting Started

### Prerequisites

- Node.js and npm/Yarn
- Xcode (for iOS build/run)
- Android Studio / Android SDK (for Android build/run)
- [Expo CLI](https://docs.expo.dev/more/expo-cli/)

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the project locally:
   - For iOS:
     ```bash
     npm run ios
     ```
     *(Note: Ensure you have CocoaPods installed and run `pod install` in the `ios` directory generated by prebuild if required.)*

   - For Android:
     ```bash
     npm run android
     ```

## Usage

When launching the app, you will see a simple UI that includes:
- Current torrent **Status** (Idle, Starting..., Downloading...)
- Download **Progress** (%)
- A **"Start Download"** button which triggers a test magnet link download (e.g., an Ubuntu ISO) saved directly to the device's document directory.
